# Dungeon Rush #
Dungeon Rush is an interactive 3D game made in Godot version 4.6. Being strategic, quick thinking, and fast moving.

### Features 🕵️ ###
- Storytelling Plot
- Code Finding Games
- Moving challenges

### Controls ⬆️ ###
- WASD or Arrows keys to move
- Space to jump
- Mouse to adjust screen
- ESC to pause
- E to interact with objects
  
### How To Play 🎮 ###
- Download .exe
- Open Itch.IO and search up Fishyyyyyyy_life
- Click the game "Dungeon Rush"
- Follow the Instalation Insturctions, press the "Download button", extract the zip file, and run index.exe inside the extracted folder.
  Make sure both index.exe and index.pck are in the same folder when you run it.

### Goal 🪙 ###
- Player has to complete a set of challenges to reach the end of the game

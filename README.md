# Dungeon Rush #
Dungeon Rush is an interactive 3D game made in Godot version 4.6. Being strategic, quick thinking, and fast moving.

### Features 🕵️ ###
- Storytelling Plot
- Code Finding Games
- Moving challenges

### Controls ⬆️ ###
- WASD or Arrows keys to move
- Space to jump
- Mouse to adjust screen
- ESC to pause
- E to interact with objects
  
### How To Play 🎮 ###
- Playable in browser
- Open Itch.IO and search up Fishyyyyyyy_life
- Click the game "Dungeon Rush"
  
### Goal 🪙 ###
- Player has to complete a set of challenges to reach the end of the game
